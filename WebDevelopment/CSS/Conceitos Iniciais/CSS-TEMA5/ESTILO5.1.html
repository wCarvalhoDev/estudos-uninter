posição: estática;

Um elemento com posição: estático; é sempre posicionado de acordo com o fluxo normal da pagina:

Aqui - position: static;

posição: relativa

Um elemento com posição: relativa; está posicionado em relação a sua posição normal:

Aqui - position: relative;
